# 1. 第一章 马克思主义自然观

## 1.1 内容总结

### 1）马克思主义自然观的形成

#### 1.1）朴素唯物主义自然观

- 观点：
  - 自然界的本原是某一种物质或某几种物质或某种抽象的东西
  - 自然界不是静止不变的，而是运动变化的，”处于永恒的产生和消灭中，处于不断的流动中，处于无休止的运动和变化中“
  - 生物是不断进化的，并在其中分化出了人
    - 古代中国朴素唯物主义自然观主要观点 / 古希腊朴素唯物主义自然观主要观点 比较
- 特征：整体性和直观性，思辨性和臆测性，自发性和不彻底性
- 渊源：原始宗教神话自然观
- 基础：自然哲学
- 作用：马克思主义自然观形成的思想渊源，近代自然科学发展的历史渊源  ，当代生态自然观和生态文明观形成的思想渊源
  - 中国 / 古希腊
- 缺陷：不能彻底地坚持唯物主义，不能满足民众需要，不能科学地说明自然界
- 演变：中世纪宗教神学自然观，文艺复兴时期的自然观

#### 1.2）机械唯物主义自然观

- 观点：
  - 自然界由物质构成，物质由不可再分的微粒构成
  - 自然界具有绝对不变性，自然物和时间，空间都是不变的
  - 自然界的物质运动是受外力作用的，遵循因果规律的机械运动，宇宙的过程可以用简单的数学方程式表示
  - 自然界的安排受到上帝的”目的性“支配
  - 以形而上学的思维方式认识自然界
  - 人与自然界都是机器，并且是分立的
- 特征：机械性，不彻底性，形而上学性
- 渊源：数本原论，原子论，地动说，位移运动说
- 基础：科学基础：经典力学理论。 技术基础：工厂手工业代替家庭手工业，。。。的技术发展与中国。。。技术传入
- 作用：为辩证唯物主义自然观的形成创造了条件 / 提供了前提 / 起到了过渡作用
- 缺陷：以机械决定论认识自然界，以因果决定论看待自然界，以孤立和静止的方式研究自然界

#### 1.3）辩证唯物主义自然观

- 观点：
  - 自然界是先在的和历史的自然界
  - 自然界是相互联系和变化发展的自然界
  - 人是自然界的一部分，实践是人类认识和改造自然界的活动
  - 用辩证思维方式认识自然界
- 特征：实践性，历史性，辩证性，批判性
- 渊源：古希腊哲学，德国古典哲学
- 基础：科学基础：星云假说，人工合成尿素，电磁场理论，细胞学说。技术基础：蒸汽机技术革命，电力技术革命
- 作用：实现了自然观发展史上的革命性变革，为马克思主义自然观的形成奠定了理论基础，为自然科学的发展奠定了方法论基础，为自然科学和社会科学的融合奠定了理论基础，为解决生态环境问题提供世界观和方法论，成为系统自然观人工自然观和生态自然观形成的思想渊源。

### 2）马克思主义自然观的发展

#### 2.1）系统自然观

>系统自然观是关于自然界的存在及其演化的观点，是以系统科学等为基础，对自然界系统的存在方式和演化规律的概括和总结。

- 观点：
  - 自然界是系统的自然界
    - 自然界是以系统的方式存在的，是简单性与复杂性，构成性与生成性，确定性与随机性相统一的物质系统
    - 系统是由若干要素通过非线性相互作用构成的整体，它具有开放性，动态性，整体性和层次性等特点
  - 自然界是演化循环的自然界
    - 自然界的演化是不可逆的，分叉和突现是其演化的基本方式，开放性，远离平衡态，非线性作用和涨落等构成其演化的机制
    - ”自然界经历了混沌-有序-新的混沌-新的有序的循环发展过程“

- 特征：系统性，复杂性，演化性，广义性
- 渊源：古代希腊（赫拉克利特，亚里士多德）和中国的系统思想，近代辩证法

- 基础：
  - 科学基础：系统论，控制论，信息论。耗散论，协同论，突变论，超循环论。分形论，混沌论。
  - 技术基础：计算机，自动装置，自动调节机械等硬件。系统技术中的新的理论成果和学科的软件。

- 作用：丰富和发展了马克思主义物质论 / 认识论和方法论 / 价值论 / 实践论
  - 。。。

#### 2.2）人工自然观

>人工自然观是关于人类改造自然姐的总的观点，是以现代科学技术成果为基础，对人工自然界的存在，创造与发展规律及其与天然自然界的关系进行的总结和概括。

- 观点：
  - 人工自然界是人类运用科学技术创造的系统的自然界，它具有目的性，实践性，价值性等特征
  - 人工自然界和人化自然界皆来源于天然自然界，他们三者通过相互交换物质，能量和信息不断地演化着
  - 人工自然界通过“自复制” ”自催化“ ”自反馈“ 等机制，从简单到复杂， 从低级到高级”螺旋式“地演化着
  - 遵循自然和社会发展规律，树立尊重自然，顺应自然，保护自然的生态文明理念。。。

- 特征：主体性，能动性，价值性

- 渊源：人工客体，五行说
- 基础：
  - 科学基础：系统科学，生态科学。哲学，美学，伦理学，法学
  - 技术基础：采取，加工，控制。大数据，云计算，互联网

- 作用：丰富和发展了历史唯物主义自然观。实现了唯物论，辩证法，实践论和价值论的统一。有助于实现人工自然界和天然自然界的统一。

#### 2.3）生态自然观

>生态自然观是关于人与生态系统辩证关系的总的观点。是在全球生态危机的背景下，依据生态科学和系统科学的成果，对人类和自然界关系进行的概括和总结。

- 观点：

  - 生态自然界系统具有整体性，多样性，层次性，开放性，动态性，自适应性和自组织性等特征，他是多样性和整体性，平衡和非平衡的统一，天然自然界和人工自然界的统一。
  - 通过从自然界的人工化转向生态化，从非生态型人工自然界转向生态人工自然界，”人类必须敬畏自然，尊重自然，顺应自然，保护自然“ ”践行绿水青山就是金山银山“ ”着力推进人与自然和谐共生“ ”动员全社会力量推进生态文明建设，共建美丽中国“ 实现任何自然界的可持续发展。
  - 贯彻落实新发展理念，”加大生态系统保护力度“ ”改革生态环境监管体制“ 实施节能减排和发展低碳经济， 构建和谐社会，构建生态文明。

  >生态自然界是系统的 / 和谐的 / 演化的

- 特征：全球性，批判性，和谐性
- 渊源：敬天畏天，天人合一（斯宾诺莎，卢梭，马克思，恩格斯）
- 基础：
  - 科学基础：生态学，生物学，数学
  - 技术基础：生态技术，生物技术

- 作用：
  - 丰富和发展了马克思主义自然观
  - 有助于深入理解新发展理念（创新，协调，绿色，开放，共享）
  - 有助于生态文明建设

### 2）科学技术的本质结构

#### 2.1）科学技术的本质特征

- 科学的本质特征：
  - 科学（主要指狭义科学）是在人类探索自然实践活动基础上的理论化，系统化的知识体系，科学知识是人在与大自然接触的过程中获得的对自然的认识
  - 科学是产生知识体系的认知活动，科学的任务就是发现事实，解释客观事物的规律
  - 科学是一种社会建制，即一项成为现代社会组成部分的社会化事业
  - 科学是一种文化现象，是人类文化中最基本的组成部分
- 技术的本质特征：
  - 技术是人类为了满足自身需要，在实践活动中根据实践经验或科学原理所创造发明的各种手段和方式方法的总和。
  - 主要体现在两个方面：一是技术活动，狭义的技术是指人类在利用自然，改造自然的劳动过程中所掌握的方法和手段；广义的技术是指人类改造自然，改造社会和改造人类资深的方法和手段。二是技术成果，包括技术理论，技能技巧，技术工艺与技术产品

- 区别与联系：
  - 科学在本质上体现了人对自然的理论和实践关系，具有客观性和实证性，探索性和创造性，通用性和共享性，现代科学通过技术体现其特征。科学是一般生产力，必须和直接的生产过程相结合才能转化为现实的生产力。
  - 技术在本质上体现了人对自然的实践关系，是人的本质力量的展现，属于直接生产力，是自然性和社会性，物质性和精神性，中立性与价值性，主体性和客观性，跃迁性和累积性的统一。
  - 科学是间接生产力，技术是直接生产力
  - 科学回答”是什么，为什么“技术回答”怎么做“
  - 科学不可预见，技术可预见
  - 科学进步的标准在于能否完善科学理论，技术提高的标准在于能否生产出新的和更好的产品
  - 科学提供物化的可能，技术提供物化的现实
  - 科学是发现，技术是发明
  - 科学是创造知识的研究，技术是综合利用知识与需要的研究
  - 区别科学与技术的目的，不是见他们分开，而是要更好的统一考虑。注重技术时要想到科学，注重科学时要考虑技术。对于科学来说，技术是科学的延申，对于技术来说，科学是技术的升华。

### 3）科学技术的发展模式及动力

-  马恩观点：
  - 在科学发展模式问题上，马克思主义认为，科学的发展表现为渐进与飞跃的统一，继承与创新的统一，分化与综合的统一
  - 在技术发展动力上，马克思主义认为，社会需求与技术发展水平之间的矛盾是技术发展的基本动力，技术目的和技术手段之间的矛盾是技术发展的直接动力，科学技术的交叉融合是技术发展的重要推动力
- 西方观点：

  - 社会建构论：技术的发展过程中，社会因素起决定作用
- 技术自主论和社会构建论都看到了技术发展的某一方面的动力，忽视或低估了其他方面动力的作用，存在片面性。
